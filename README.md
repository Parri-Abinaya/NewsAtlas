# 🌍 News Atlas

**A production-ready interactive world news intelligence platform.**

Click any country on the map to instantly view live news, weather, currency rates, market data, and AI-generated country summaries.

---

## ✨ Features

- **🗺 Interactive World Map** — SVG world map with click detection, hover highlights, zoom/pan, and country tooltips
- **📰 Live News Feed** — Country-specific and global headlines via GNews API
- **🌤 Live Weather** — Current conditions for each country's capital (OpenWeatherMap)
- **💱 Currency Exchange** — Real-time exchange rates via ExchangeRate API
- **📊 Global Markets** — Stock indices, currency pairs, and commodities
- **✦ AI Summaries** — Country intelligence briefs generated by Claude AI
- **🔍 Country Search** — Real-time search with debounced filtering
- **🌙 Dark/Light Theme** — Toggle with persistence
- **💾 Save Countries** — Save and retrieve favorite countries (MongoDB)
- **📩 Feedback Form** — Validated form with MongoDB storage
- **⚡ Response Caching** — Server-side caching to minimize API calls

## 🚀 Installation & Setup

### Prerequisites
- Node.js v18+
- MongoDB (local or [MongoDB Atlas](https://www.mongodb.com/atlas))
- API keys (see below)

### 1. Clone & Install

```bash
git clone https://github.com/yourusername/news-atlas.git
cd news-atlas
npm run install:all
```

### 2. Configure Environment Variables

**Server** — copy and fill in `server/.env`:
```bash
cp server/.env.example server/.env
```

```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/newsatlas
GNEWS_API_KEY=your_gnews_key
OPENWEATHER_API_KEY=your_openweather_key
EXCHANGE_API_KEY=your_exchangerate_key
ANTHROPIC_API_KEY=your_anthropic_key
```

**Client** — copy and fill in `client/.env`:
```bash
cp client/.env.example client/.env
```

```env
REACT_APP_API_URL=http://localhost:5000/api
```

> **Note:** The app works without API keys — it falls back to realistic mock data for all external sources.

### 3. Run the App

```bash
# Run both frontend and backend concurrently
npm run dev

# Or separately:
npm run server   # Express server on :5000
npm run client   # React app on :3000
```

Open [http://localhost:3000](http://localhost:3000)

## 🔗 API Endpoints

```
GET  /api/health                    Health check
GET  /api/country/search?q=france   Search countries
GET  /api/country/featured-news     Global headlines
GET  /api/country/:code             Full country data (news+weather+currency+facts+AI)
GET  /api/markets                   Market indices + currencies + commodities
POST /api/feedback                  Submit feedback { name, email, message }
GET  /api/feedback                  List feedback
POST /api/saved                     Save a country { countryCode, countryName }
GET  /api/saved                     Get saved countries
DEL  /api/saved/:code               Remove saved country
```
